Project Overview

Visual Concept

sunfreeclub의 브랜드 기획과 디자인 방향을 보여주는 포트폴리오 섹션입니다.
태양, 산, 등고선, 유틸리티 웨어의 감각을 시각 언어로 정리했습니다.

Typography

SUNFREECLUB

Type Note / Helvetica Neue

영문 타이틀은 Helvetica Neue 계열의 산세리프를 사용해 미니멀하고 테크니컬한 인상을 만듭니다.

Graphic Motif

Sun, trail line, contour pattern, and utility grid are used as repeatable brand motifs.

Vision

Brand Overview

sunfreeclub은 삶의 나침반이 되어주는 태양과 함께할 수 있는 브랜드를 만듭니다.
도시와 자연의 경계에서 작동하는 절제된 형태와 기능적인 제품으로 '자유로운 빛을 입는다'는 감각을 제안합니다.
태양은 그 누구에게도 값을 제시하지 않습니다. 그저 나가기만 하면 되니까요.
